2017/10/21少女前線--其他點擊

是說最近無聊想把少女前線拆開來看看裡面到底是什麼順便拿一下高清大圖和立繪…雖然最後都是一堆1024x1024的而已

拆包檔案其實分兩部分，一部分是apk裡面有的資料，而另一部分就是安裝好遊戲後下載來的資料，還有語言。在這篇裡面我們會先進行apk的拆包來熟識一下我們將會用到的工具和界面

不過拆之前還是要說一下拆包一直都是一個道德爭議的問題(?)，此文僅僅是為了滿足學習需求而打的要使用後果自負(?)

拆包需求

少女前線的apk
這裡我會建議直接在陸服的官網下載，基本上因為常駐活動和活動回想的關係以前出現過的CG回想等等都能在最新版本找到。
Unity Studio
由於少女前線是由Unity寫成的你拆包了你就會知道為什麼我知道，這是一個由某位大大寫的open source Unity拆包工具，少前現在的構成比較簡單所以大部分的data我們都可以通過Unity Studio來拆包獲得

如果圖片的話只要上面兩個就好，如果要拆BGM音樂等我們還需要這兩個
VGMToolbox
和
hcaDecoder

拆開apk

首先我們在官網下載apk，下載回來應該大概是這個樣子的：

我們先用Winrar打開apk然後把所有東西解壓出來，得到的會是這樣的東西

而我們需要的只有assets/bin/Data裡面的東西

當然這只是因為我們只要拆圖片資源之類的出來所以才只需要這些東西，其他對遊戲運行也是不可缺少的有興趣可以去繼續研究一下
然後為了Unity Studio的使用方便我們先把Managed和Resources這兩個資料夾都丟走，裡面的東西我們不會用到所以刪了也沒什麼關係

Unity Studio 拆包

打開你的Unity Studio，在左上角的File選擇Load folder, 找到你剛才的拆的Data然後選擇Open，然後等他自己跑就好
跑完之後會出現以下這些東西，因為太麻煩了所以我們不管他而是轉去看看Asset List有點什麼


為了方便起見，按一下Type讓我們用類別來排列更好的看
在這吐糟一句雲母改名真的毫無規矩邏輯，簡直是spaghetti naming
這裡主要有4個種類的檔案可以可以，其中mesh和Shader是和UI相關的這個我沒什麼興趣去弄，除非你想自己合成UI還有2Dlive之類的不然沒什麼必要去碰。TextAsset有兩個類別的文件，可能是格式類似的原因Unity Studio沒把他分開，但我們可以用副檔名來分辨。沒有副檔名的基本上都是文字，例如UI內容之類的，而.acb多是音訊內容，之後我們會說說該怎麼拆出來。而剩下的Texture2D就基本上是圖片，還有製造各種效果UI需要的素材圖，也就是我們主要的目標了。

拆CG


由於這個奇怪的CG命名方式，我們最好還是按下去看看到底是什麼圖我們想不想要再extract出來。

選好需要的檔案之後(請善用Shift和Ctrl來多選)，我們去找Export然後export Selected assets，再選擇export到哪裡，然後Unity Studio就會自動跑把Assets丟到指定的資料夾了。
這裡有兩個注意的地方，Unity Studio本身不是一個十分穩定的軟體，如果export大量的assets(例如export all assets)他很容易會掛掉，要避免的話最好還是選擇需要的一批批的export。不過雖然說是這樣幾十個assets的話問題也不算很大，不過這個有各方面因素影響就不一一實驗了。另外的是除了在assets這邊選也可以回到Scene Hierarchy選，不過我個人覺得看assets這邊比較清楚明白就是了。
另外的就是選擇export的資料夾，由於Unity Studio並沒有留意太多user friendly的設計所以每一步export完他都會跳回你assets所在的資料夾，也就是說你如果想要export到外面的話那你就要不斷的選路徑了。而解決方法很簡單，由於export出來的assets都會根據Type來分類丟進不同的資料夾，所以在原本的路徑export完再手動移動就好。
Export完之後我們就可以在Texture2D裡面找到我們的圖片了，TextAsset等等別的種類也是一樣。

拆音樂

拆音樂的話首先的做法和CG一樣，不過我們的目標是TextAsset裡面有.acb作副檔名的assets不要問我為什麼是TextAsset
由於他的分類是TextAsset解壓出來之後會出現這樣的情況

請把檔案名字中的.txt去掉就能獲得.acb file了因為音樂不是我的目標所以我只是手動意思意思的拆了兩個實驗，要取得所有音樂請善用Script來改名
不過.acb到底是什麼呢？根據這裡的第二個這是一個用CRI包裝的audio file，由於我沒錢買CRI的關係我們只能用別的方法來拆出來了。
首先我們要用VGMToolbox 來把acb file解碼
打開VGMToolbox.exe，選擇VGMToolbox>Misc. Tools>Extraction Tools>Common Archives>CRI ACB/AWB Archive Extraction

把我們需要的acb file丟進去(可以同時丟一堆)然後就會解碼出一個file裝着acb，一直按進去按到最後會找到一個hca的file
hca算是一種在遊戲中比較常見的audio game data file，不過我們業餘開發比較少會用到它，常見的播放器也沒什麼辦法去播放這個所以我們需要進一步的解碼
此時我們就要用到這個hcaDecoder，不熟識Github的朋友請在網頁右方找Download

把下載完的zip解壓縮，然後找到hcaToWav

把我們需要解碼的hca檔案丟進去他就會自動幫你轉wav了，我希望沒人的電腦不能開wav吧(?)
那麼音樂解壓縮就到這裡了，如果想批量解壓建議還是善用script把檔案放一起然後批量處理，不過身為一個懶人這裡就先不弄了。

總結

在這邊我們已經把apk裡的大部分內容解出來了，而相信大家也熟悉了各種工具的使用，不過你們可能疑惑為什麼只有CG和音樂，立繪CV呢？這些檔案並不是放在apk裡而是遊戲開始之後再下載到手機的，所以要找我們需要用別的方法來找，而步驟也不少所以就不寫在這裡而是另外一篇了。
